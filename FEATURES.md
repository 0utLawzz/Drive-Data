# Features

## Main Features

- **Multi-Mode Directory Scanning:** Process All Clients, Consultants, Both directories, or any custom path in a single run.
- **Regex-Based Pattern Classification:** 13 predefined trademark document categories automatically recognized without manual sorting.
- **Flexible Output Targets:** Single tool supports local (Excel/CSV) exports, cloud (Google Sheets) uploads, or both simultaneously.

## Data Quality Features

- **Duplicate Prevention:** Before uploading to Google Sheets, the tool fetches existing records and computes a unique composite key per row to avoid re-inserting known data.
- **Quick Export Mode:** A fast-path option that exports raw directory listings without pattern matching, for rapid auditing.

## Batch & Testing Features

- **Batch Processing Limit:** Specify a maximum record count per run to safely test configurations before running on complete datasets.
